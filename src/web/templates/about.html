Funcionamiento del Sistema

Análisis Inteligente de Noticias

El sistema procesa automáticamente noticias de múltiples fuentes RSS internacionales, aplicando técnicas avanzadas de NLP para extraer ubicaciones geográficas, evaluar niveles de riesgo y categorizar tipos de conflictos mediante IA.

Correlación con GDELT

Los resultados del análisis de noticias se cotejan con la base de datos GDELT para determinar con máxima precisión las coordenadas geográficas exactas de las áreas de conflicto, validando y refinando la información.

Análisis Satelital Automatizado

Una vez identificadas las coordenadas precisas, el sistema solicita automáticamente imágenes satelitales de SentinelHub de las zonas de conflicto activo, descargando datos actualizados para análisis visual posterior.

Computer Vision Avanzado

Las imágenes satelitales se analizan mediante un modelo preentrenado con 5 datasets unificados (incluyendo 2 sintéticos de videojuegos) para detectar vehículos militares, fuego, explosiones y edificios destruidos.

Capacidades Técnicas

Tiempo Real

Monitoreo continuo 24/7 con ingesta automática cada hora, procesamiento NLP cada 2 horas y análisis histórico cada 6 horas.

Cobertura Global

Análisis multi-idioma (EN, ES, FR, DE, RU, ZH, AR) con integración de feeds GDELT, ACLED y fuentes de inteligencia geopolítica.

Alta Precisión

Validación cruzada entre múltiples fuentes, correlación GDELT y verificación satelital para máxima fiabilidad en la detección.
