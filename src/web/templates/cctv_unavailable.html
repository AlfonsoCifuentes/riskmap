Dependencias Requeridas

  • OpenCV para procesamiento de video
  • PyTorch para modelos de IA
  • Ultralytics YOLO para detección de objetos
  • FFmpeg para manipulación de streams
  • Flask-SocketIO para comunicación en tiempo real
  • yt-dlp para resolución de URLs de video
  • Celery y Redis para procesamiento en segundo plano

Instrucciones de Instalación

Para habilitar el sistema CCTV, ejecute los siguientes comandos:

# Instalar dependencias básicas
pip install opencv-python ultralytics torch torchvision
pip install flask-socketio yt-dlp ffmpeg-python
pip install celery[redis] pillow geopandas shapely
# Instalar FFmpeg (Windows)
# Descargar desde: https://ffmpeg.org/download.html
# O usar chocolatey: choco install ffmpeg
# Instalar Redis (para procesamiento en segundo plano)
# Windows: https://redis.io/download
# O usar Docker: docker run -d -p 6379:6379 redis
Volver al Inicio Dashboard Principal

Una vez instaladas las dependencias, reinicie la aplicación para activar el sistema CCTV.
